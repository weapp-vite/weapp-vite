# multi-platform-demo

`weapp-vite` 多平台示例

## 使用方式

### 开发

- `pnpm dev`（默认 weapp）
- `pnpm dev:alipay`
- `pnpm dev:tt`
- `pnpm dev:swan`
- `pnpm dev:jd`
- `pnpm dev:xhs`
- `pnpm dev:web`

- `pnpm dev:open` 可以打包并直接启动微信开发者工具

### 构建

`pnpm build`（默认 weapp）

其他平台：
- `pnpm build:alipay`
- `pnpm build:tt`
- `pnpm build:swan`
- `pnpm build:jd`
- `pnpm build:xhs`
- `pnpm build:web`

### 打开微信开发者工具

`pnpm open`

### 生成组件/页面

`pnpm g path/to/your/component`
