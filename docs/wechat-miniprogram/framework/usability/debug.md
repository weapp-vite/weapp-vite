# 调试

开发者可以借助下列工具进行小程序的调试。

- [vConsole](./vConsole.md) ：在手机上查看 `console` API 输出的日志内容和额外的调试信息。
- [Source Map](./sourceMap.md) ：还原 JS 错误堆栈
- [真机调试](https://developers.weixin.qq.com/miniprogram/dev/devtools/remote-debug-2.html) : 利用开发者工具，通过网络连接，对手机上运行的小程序进行调试，帮助开发者更好的定位和查找在手机上出现的问题。
- [实时日志](../realtimelog/README.md) ：快捷地排查小程序漏洞、定位问题
