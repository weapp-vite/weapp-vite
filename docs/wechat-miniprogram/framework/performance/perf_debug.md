# 性能与体验调试工具

为了协助开发者调试与优化小程序的性能与体验，我们提供了一些工具：

- [真机调试 2.0](./remote_debug_2.md) ：通过开发者工具调试真机运行环境。
- [开发者工具「模拟器」和「调试器」](./devtools-perf.md) ：在开发者工具「模拟器」中对小程序视图层分析和进行低端机模拟，在「调试器」环境中进行 JS 性能、内存占用分析。
- [代码质量分析面板](./quality-panel.md) ：静态分析开发者代码，给出优化建议。
- [FPS 面板](./fps_panel.md) ：展示页面当前的实时帧率
- [性能面板](./panel.md)
- [体验评分](../audits/audits.md) ：自动化检测小程序性能与体验问题，并为开发者提供优化建议。
