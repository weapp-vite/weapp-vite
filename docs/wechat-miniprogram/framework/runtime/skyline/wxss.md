## 开启默认Block布局

Skyline 下节点默认为 flex 布局，可通过以下配置切换为默认 block 布局。

<table><thead><tr><th>平台</th> <th>最低版本</th></tr></thead> <tbody><tr><td>Android</td> <td>8.0.34</td></tr> <tr><td>IOS</td> <td>8.0.36</td></tr> <tr><td>开发者工具</td> <td>Nightly Build (1.06.2304262)</td></tr> <tr><td>基础库</td> <td>2.31.1</td></tr></tbody></table>

在 `app.json` 或 `page.json` 中配置：

```json
rendererOptions: {
  "skyline": {
    "defaultDisplayBlock": true,
  }
}
```

## 开启tag选择器全局匹配

Skyline 下 tag 选择器遵循样式隔离机制，而 WebView 下不受样式隔离约束，可通过 `tagNameStyleIsolation: legacy` 配置对齐 WebView 表现，若指定 `tagNameStyleIsolation: isolated` 则遵循样式隔离机制。

<table><thead><tr><th>平台</th> <th>最低版本</th></tr></thead> <tbody><tr><td>Android</td> <td>8.0.51</td></tr> <tr><td>IOS</td> <td>8.0.51</td></tr> <tr><td>开发者工具</td> <td>Nightly Build (1.06.2409032)</td></tr> <tr><td>基础库</td> <td>3.6.0</td></tr></tbody></table>

在 `app.json` 或 `page.json` 中配置：

```json
rendererOptions: {
  "skyline": {
    "tagNameStyleIsolation": "legacy",
  }
}
```
