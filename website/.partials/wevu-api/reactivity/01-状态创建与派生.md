<WevuApiDocGroup :api-count="8" summary="创建深层、浅层、只读和派生状态，并控制响应式代理范围。" default-open title="状态创建与派生">

### `ref()` {#ref}

<!-- api-reference-details -->

**类型签名：** `typeof import('wevu')['ref']`

**运行时说明：** 依赖变化进入 Wevu 调度队列，并最终收敛为小程序 `setData` 更新；模板状态必须保持可序列化。

**示例：** 见 [本组示例](/wevu/api/reactivity#example-reactivity-state)。

- 类型入口：`Ref<T>`
- 用途：创建最常用的基础响应式引用，适合原始值或独立状态。
- 说明：读取使用 `ref.value`；在模板中会自动解包。

### 本组示例 {#example-reactivity-state}

组合深层状态、独立 Ref 和派生值时，只把可序列化数据交给模板。

```vue
<script setup lang="ts">
import { computed, reactive, readonly, ref, shallowRef } from 'wevu'

const count = ref(0)
const form = reactive({ price: 20, quantity: 1 })
const sdk = shallowRef<{ refresh: () => void }>()
const total = computed(() => form.price * form.quantity)
const publicForm = readonly(form)
</script>
```
