### 本组示例 {#example-reactivity-effects}

把相关监听放进同一 scope，可以在页面卸载或业务结束时统一停止。

```ts
import { effectScope, onScopeDispose, ref, watch, watchEffect } from 'wevu'

const count = ref(0)
const scope = effectScope()

scope.run(() => {
  const stopWatch = watch(count, value => console.log(value))
  watchEffect(() => documentTitle(count.value))
  onScopeDispose(stopWatch)
})

scope.stop()
```
