## 应用实例

### `app.mount()` {#app-mount}

<!-- api-reference-details -->

**类型签名：** `RuntimeApp['mount']`

**Vue 差异：** Vue 接收 DOM 容器；Wevu 接收可选的小程序 `MiniProgramAdapter`，并返回 `RuntimeInstance`。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

该方法建立响应式状态与小程序 `setData` 之间的适配。常规小程序入口由 `createApp()` 自动注册，只有测试或自定义适配器场景通常需要显式调用。

### `app.unmount()` {#app-unmount}

<!-- api-reference-details -->

**类型签名：** `RuntimeApp['unmount']`

**Vue 兼容说明：** 都用于停止应用并执行清理，但 Wevu 不涉及 DOM 根节点移除。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

重复调用是安全的；已执行的应用级清理不会再次运行。

### `app.onUnmount()` {#app-onunmount}

<!-- api-reference-details -->

**类型签名：** `RuntimeApp['onUnmount']`

**Vue 兼容说明：** 调用方式与 Vue 一致；Wevu 用它管理小程序应用级插件和资源清理。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

传入值必须是函数。应用已经卸载时，新登记的清理函数会立即执行。

### `app.use()` {#app-use}

<!-- api-reference-details -->

**类型签名：** `RuntimeApp['use']`

**Vue 兼容说明：** 支持函数插件和带 `install()` 的对象插件，并忽略同一插件的重复安装；插件接收的是 Wevu `RuntimeApp`。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

插件中的 hook 仍需在同步初始化阶段注册，不能在 `await` 之后补注册生命周期。

### `app.provide()` {#app-provide}

<!-- api-reference-details -->

**类型签名：** `RuntimeApp['provide']`

**Vue 兼容说明：** 调用方式与 Vue 一致；依赖通过 Wevu 的页面/组件上下文链传递，而不是 DOM 组件树。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

返回应用实例，可与 `app.use()` 链式调用。

### `app.config` {#app-config}

<!-- api-reference-details -->

**类型定义：** `AppConfig`

**Vue 差异：** Wevu 当前只公开 `globalProperties`，不提供 Vue Web 的错误处理器、编译器选项、性能标记和选项合并策略。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

不要假设 Vue `app.config` 的其他字段在 Wevu 中存在。

### `app.config.globalProperties` {#app-config-globalproperties}

<!-- api-reference-details -->

**类型定义：** `Record<string, any>`

**Vue 兼容说明：** 用于向组件公开共享属性；最终访问发生在 Wevu 组件公共代理上。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

业务状态仍应优先放入 store 或 provide/inject，避免把该对象当成无边界全局状态容器。

### `app.version` {#app-version}

<!-- api-reference-details -->

**运行时值：** `string`

**Vue 兼容说明：** 与 Vue 一样用于读取当前框架版本，值对应 Wevu 包版本。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

该字段为只读版本信息，不用于运行时能力检测。

### `version` {#version}

<!-- api-reference-details -->

**运行时值：** `string`

**Vue 兼容说明：** `wevu` 根入口直接导出版本字符串，与 `app.version` 保持一致。

**示例：** 见 [本组示例](/wevu/api/core#example-app-instance)。

适合在诊断信息中展示，不要根据字符串比较切换核心业务逻辑。

### 本组示例 {#example-app-instance}

应用实例适合承载插件安装、全局依赖和应用级清理。

```ts
import { createApp, version } from 'wevu'

const app = createApp({})

app
  .use(runtimeApp => runtimeApp.provide('apiBase', '/api'))
  .onUnmount(() => console.info('dispose app resources'))

console.info(version, app.version)
```
