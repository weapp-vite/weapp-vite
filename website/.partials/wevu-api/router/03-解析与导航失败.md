### `resolveRouteLocation()` {#resolveroutelocation}

<!-- api-reference-details -->

**类型签名：** `typeof import('wevu/router')['resolveRouteLocation']`

**运行时说明：** 解析和守卫在 JavaScript 层执行，真正跳转仍由小程序 Router 完成，因此页面栈、tabBar 和宿主失败回调是最终边界。

**示例：** 见 [解析与导航失败共用示例](/wevu/api/router#router-examples)。

将字符串或位置对象解析为标准路由位置，可传入当前路径处理相对地址。
