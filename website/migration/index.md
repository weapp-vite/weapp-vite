# 迁移

如果你是在新项目中直接使用最新版本，通常无需阅读本页。这里主要服务两类场景：

- 现有原生小程序项目，希望逐步迁移到 `weapp-vite` / `wevu`
- 已经在线上运行旧版本 `weapp-vite`，需要按版本跨度升级

推荐阅读顺序：

1. [从原生小程序迁移到 Weapp-vite / Wevu](../wevu/migration/from-native-to-vue-sfc.md)
2. [从 v5 升级到 v6](./v6.md)
3. [从 v4 升级到 v5](./v5.md)
4. [从 v3 升级到 v4](./v4.md)
5. [从 v2 升级到 v3](./v3.md)
