# 教程总览

这一套教程的目标是：用 **weapp-vite** 把小程序工程化跑顺；用 **Vue SFC**（本教程称作 _wuve_：weapp-vite 内置的 Vue SFC 编译链路）写页面/组件；用 **wevu** 在小程序运行时获得 Vue 3 风格的响应式与生命周期能力。

## 你会在这里得到什么

- 从 0 到上线的完整目录（按“概念 → 写法 → 边界 → 排错”组织）。
- 覆盖常见小程序业务形态：页面、组件、表单、列表、分包、分享、滚动、上传下载等。
- 与仓库现有文档互相链接：本教程更偏“体系化串讲”，细节参考走 `/guide/`、`/config/`、`/wevu/`。

## 两条阅读路线

- **30 分钟跑通路线**：`/handbook/getting-started` → `/handbook/sfc/` → `/handbook/wevu/` → `/handbook/debugging`
- **进阶实战路线**：按左侧目录顺序通读（并配合示例工程一起跑）

## 从哪开始

- 第一次接触：先看 `/handbook/reading-guide`
- 直接开写：从 `/handbook/getting-started` 开始
- 只想系统学 SFC：从 `/handbook/sfc/` 开始
- 只想系统学 wevu：从 `/handbook/wevu/` 开始
