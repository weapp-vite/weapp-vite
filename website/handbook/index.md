# 教程总览

这套教程想帮你把“小程序工程化 + Vue SFC + 运行时”这条链路一次性跑顺，核心就三件事：

- 用 **Weapp-vite** 把项目开发/构建起来
- 用 **Vue SFC** 写页面和组件
- 用 **Wevu** 在运行时获得 Vue 3 风格的响应式与生命周期能力

和 `/guide`、`/config` 的区别是：

- `/guide` 偏功能入口，回答“这个能力怎么开”
- `/config` 偏字段索引，回答“这个配置项怎么配”
- `/handbook` 偏学习路径，回答“我应该按什么顺序建立完整心智模型”

## 你会在这里得到什么

- 从 0 到上线的一条完整路线，按“概念 → 写法 → 边界 → 排错”组织。
- 覆盖常见小程序业务：页面、组件、表单、列表、分包、分享、滚动、上传下载等。
- 和仓库现有文档互相跳转：本教程偏“串起来讲清楚”，字段与 API 细节再去 `/guide/`、`/config/`、`/wevu/` 查。

## 两条阅读路线

- **30 分钟跑通路线**：`/handbook/getting-started` → `/handbook/sfc/` → `/handbook/wevu/` → `/handbook/debugging`
- **进阶实战路线**：按左侧目录顺序通读（并配合示例工程一起跑）

## 从哪开始

- 第一次接触：先看 `/handbook/reading-guide`
- 直接开写：从 `/handbook/getting-started` 开始
- 只想系统学 SFC：从 `/handbook/sfc/` 开始
- 只想系统学 Wevu：从 `/handbook/wevu/` 开始
