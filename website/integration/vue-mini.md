# Vue-mini 集成

## vue-mini 介绍

Vue Mini 是一个基于 Vue 3 的小程序框架，你可以用 Vue 3 的响应式和组合式 API 来开发小程序。

文档：<https://vuemini.org/>

## 集成

在 `weapp-vite` 中接入 `vue-mini` 的步骤很简单：安装依赖后按官方文档使用即可。

```sh
pnpm i @vue-mini/core
```

然后你就可以跟随文档，使用 `vue-mini` 的所有特性了！
