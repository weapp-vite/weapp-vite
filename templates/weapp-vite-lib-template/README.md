# weapp-vite-lib-template

`weapp-vite` 组件库（lib 模式）模板，包含原生小程序组件与 Vue SFC 组件示例（script / script setup / 组合）。

## 使用方式

### 开发调试（小程序模拟器）

- `pnpm dev`

### 构建应用

`pnpm build`

### 构建组件库

`pnpm build:lib`

默认输出到 `dist-lib`。

### 打开微信开发者工具

`pnpm open`

### 生成组件/页面

`pnpm g path/to/your/component`

## AppID

请将 `project.config.json` 中的 `appid` 替换为你自己的小程序 AppID。
